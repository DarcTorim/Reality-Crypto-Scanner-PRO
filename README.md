 Reality-Crypto-Scanner-PRO
Высокоскоростной сканер подсетей для поиска зашифрованных .RU доменов, идеальных для маскировки протокола Reality
 
Профессиональный многопоточный сканер подсетей для поиска идеальных доменов под маскировку протокола **Reality (VLESS/Xray)**.

Инструмент автоматически сканирует IP-адреса, проверяет популярные веб-порты, извлекает домены из SSL-сертификатов и HTTP-заголовков, а затем классифицирует их по пригодности для маскировки.

 ✨ Особенности

*   **🚀 Высокая скорость:** Многопоточная архитектура (ThreadPool) для быстрого сканирования целых подсетей `/24`.
*   **🔒 Анализ шифрования:** Четкое разделение доменов на **Зашифрованные (SSL)** и **Незашифрованные (HTTP)**. Для Reality отбираются только те, что имеют валидный сертификат.
*   **🇷🇺 Приоритет .RU:** Автоматическое выявление российских доменов и маркировка их флагом приоритета.
*   **🛡️ Фильтр брендов:** Умное исключение известных глобальных брендов (Google, Microsoft, Cloudflare), которые не подходят для скрытной маскировки.
*   **📊 Глубокий поиск:** Сканирование расширенного списка портов (80, 443, 8080, 8443, 2053, 2083, 2087, 2096 и др.), где часто размещаются панели управления и альтернативные веб-сервисы.
*   **💾 Live Excel Export:** Результаты записываются в таблицу Excel в реальном времени. Файл можно открыть и наблюдать за процессом заполнения.
*   **🎨 Удобный интерфейс:** Цветной вывод в консоль с подсветкой важных находок.
